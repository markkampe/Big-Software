Class Dice

source code

object --+
         |
        Dice

This class supports formula based dice rolls. The formulae can be described in a few (fairly standard) formats:

  • DnD style ... D100, 3D6, 3D6+4
  • ranges ... 4-12
  • simple numbers ... 50
